# Projeto Netflix - Imersão Front-End com a Alura
Este projeto foi desenvolvido durante a Imersão Front-End da Alura. A proposta consiste na recriação da interface da Netflix, com o auxílio do GitHub Copilot integrado ao Visual Studio Code, utilizado para apoiar o desenvolvimento do código. Eu optei por utilizar SCSS no lugar de CSS puro.

## Tecnologias
- `HTML`
- `SCSS`
- `JavaScript`
- `Copilot`

## Características
- Seleção de perfis de usuário, com redirecionamento dinâmico para a página de catálogo correspondente ao perfil escolhido.

## Processo
O projeto foi construído ao longo de quatro aulas, seguindo uma abordagem incremental:
- Aula 1: Estruturação da página utilizando HTML
- Aula 2: Estilização da interface com CSS
- Aula 3: Implementação de responsividade para diferentes tamanhos de tela
- Aula 4: Integração da página de catálogo e desenvolvimento da lógica em JavaScript para seleção e redirecionamento de perfis
